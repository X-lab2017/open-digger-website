# GitHub

## 数据来源

OpenDigger 使用 [GHArchive](https://www.gharchive.org/) 作为数据源，采集 GitHub 全域日志数据并使用 [ClickHouse](https://github.com/ClickHouse/ClickHouse) 云服务作为数据服务基础设施。

数据采集、清洗与入库的相关代码目前并未开源至 OpenDigger 项目，作为定时任务每小时运行并导入数据到数据库。

## 数据缺失
